세부 내용

  1. 머신러닝이란?
    • 개념 정의

      머신러닝은 컴퓨터가 명시적으로 프로그래밍되지 않고도 데이터를 통해 스스로 학습하고 개선할 수 있는 기술입니다. 쉽게 말해, 컴퓨터가 경험을 통해 학습하는 방식입니다.

    • 작동 원리

      컴퓨터는 대량의 데이터를 분석하고 그 안에서 패턴을 찾아내어, 미래의 결과를 예측하는 방식으로 작동합니다. 예를 들어, 과거의 정보를 바탕으로 새로운 데이터를 예측합니다.

    • 실제 활용 사례
      • 넷플릭스의 영화 추천 시스템
      • 스마트폰의 얼굴 인식 기능
      • 자율주행 자동차의 장애물 감지 기능
  2. 머신러닝 학습 방법
    • 지도 학습 (Supervised Learning)
      • 정의: 이미 레이블이 붙은 데이터로 학습하여 예측하는 방법입니다. 이 방법은 주어진 입력에 대해 올바른 출력을 학습합니다.
      • 예시: 스팸 메일 분류, 주택 가격 예측
      • 주요 알고리즘: 선형 회귀, 로지스틱 회귀, 결정 트리
      • 알고리즘 예시
        • KNeighborsClassifier: 주어진 데이터 포인트와 가장 가까운 K개의 이웃을 기준으로 분류합니다.
        • KNeighborsRegressor: KNN 알고리즘을 사용하여 연속적인 수치를 예측하는 회귀 분석을 수행합니다.
        • LinearRegression: 입력 변수와 출력 변수 간의 선형 관계를 모델링합니다.
        • LogisticRegression: 이진 분류 문제를 해결하기 위해 사용되며, 확률을 기반으로 결과를 예측합니다.
        • DecisionTreeClassifier: 데이터를 분할하여 트리 구조로 학습하며, 각 분할은 특정 조건에 따라 이루어집니다.
        • RandomForestClassifier: 여러 개의 결정 트리를 조합하여 보다 정확한 예측을 제공합니다.
        • SupportVectorMachine (SVM): 데이터를 분류하기 위해 최적의 경계선을 찾는 방법입니다.
        • GradientBoostingClassifier: 여러 약한 학습기를 결합하여 강력한 예측 모델을 만드는 방법입니다.
    • 비지도 학습 (Unsupervised Learning)
      • 정의: 레이블이 없는 데이터에서 숨겨진 패턴을 찾아내는 방법입니다. 이 방법은 데이터의 구조를 이해하는 데 도움을 줍니다.
      • 예시: 고객 세분화, 추천 시스템
      • 주요 알고리즘: K-means 클러스터링, PCA
      • 알고리즘 예시
        • KMeans: 데이터를 K개의 클러스터로 나누어 군집화합니다.
        • DBSCAN: 밀도 기반 클러스터링 방법으로, 밀도가 높은 영역을 찾아 클러스터를 형성합니다.
        • Hierarchical Clustering: 계층 구조로 클러스터를 형성하여 데이터 간의 유사성을 시각화합니다.
        • PCA (Principal Component Analysis): 고차원 데이터를 저차원으로 변환하여 데이터의 분산을 최대화합니다.
        • Gaussian Mixture Model (GMM): 데이터가 가우시안 분포의 혼합으로 이루어져 있다고 가정하고 클러스터링합니다.
    • 강화 학습 (Reinforcement Learning)
      • 정의: 환경과 상호작용하면서 최적의 행동을 학습하는 방법입니다. 에이전트는 보상을 통해 학습합니다.
      • 예시: 알파고, 로봇 제어 시스템
      • 핵심 개념: 보상과 페널티를 통해 학습하여 최적의 결과를 도출합니다. 에이전트는 행동을 선택하고 결과에 따라 보상을 받습니다.
      • 알고리즘 예시
        • Q-Learning: 상태-행동 쌍에 대한 Q값을 업데이트하여 최적의 정책을 학습합니다.
        • Deep Q-Networks (DQN): 신경망을 사용하여 Q값을 근사합니다.
        • Policy Gradient Methods: 정책을 직접 최적화하여 행동을 결정합니다.
        • Actor-Critic Methods: 정책과 가치 함수를 동시에 업데이트하여 학습합니다.
  3. 파이썬으로 간단한 머신러닝 코드 작성하기
    • 학습 데이터 준비하기

      엑셀 파일로 집값 예측을 위한 간단한 데이터셋을 준비합니다. 예를 들어, 면적과 방 개수를 포함한 데이터를 사용합니다.

    • 파이썬 코드 작성하기
      • 사이킷런 설치 
        pip install scikit-learn pandas numpy
      • 데이터 로드 및 모델 생성 
        
import pandas as pd
import numpy as np
from sklearn.neighbors import KNeighborsRegressor
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from sklearn.metrics import mean_squared_error, r2_score
import matplotlib.pyplot as plt

def analyze_data(file_path, feature_column, target_column):
    # 1단계: 데이터 로드
    try:
        data = pd.read_csv(file_path, encoding='UTF-8')
    except FileNotFoundError:
        print("파일을 찾을 수 없습니다. 파일 경로를 확인하세요.")
        return
    except Exception as e:
        print(f"데이터를 로드하는 중 오류가 발생했습니다: {e}")
        return

    # 데이터에 열이 존재하는지 확인
    if feature_column not in data.columns or target_column not in data.columns:
        print(f"'{feature_column}' 또는 '{target_column}' 열이 데이터에 없습니다. 열 이름을 확인하세요.")
        return

    # 2단계: 데이터 준비
    try:
        X = data[[feature_column]].astype(int).values  # 특성 열을 정수로 변환
        y = data[target_column].astype(float).values  # 목표 열을 실수로 변환
    except ValueError:
        print("열 데이터를 변환하지 못했습니다. 열 데이터 형식을 확인하세요.")
        return

    # 데이터 표준화
    scaler = StandardScaler()
    X_scaled = scaler.fit_transform(X)

    # 데이터 분리 (80% 학습, 20% 테스트)
    X_train, X_test, y_train, y_test = train_test_split(X_scaled, y, test_size=0.2, random_state=42)

    # 3단계: 모델 학습 및 예측
    model = KNeighborsRegressor(n_neighbors=5)
    model.fit(X_train, y_train)
    predictions = model.predict(X_test)

    # 4단계: 결과 시각화
    plt.rc('font', family='Malgun Gothic')  # 한글 폰트 설정
    plt.figure(figsize=(10, 6))

    # 실제 값(파란색)과 예측 값(빨간색)을 그래프에 표시
    original_feature_values = scaler.inverse_transform(X_test)
    plt.scatter(original_feature_values, y_test, color='blue', label='실제 값', alpha=0.6)
    plt.scatter(original_feature_values, predictions, color='red', label='예측 값', alpha=0.6)

    plt.title('KNN을 사용한 예측 결과')
    plt.xlabel(feature_column)
    plt.ylabel(target_column)
    plt.legend()
    plt.grid()
    plt.show()

    # 5단계: 성능 평가
    r2 = r2_score(y_test, predictions)
    mse = mean_squared_error(y_test, predictions)
    print(f"모델 성능 평가:\n - R² (결정 계수): {r2:.4f}\n - MSE (평균 제곱 오차): {mse:.4f}")

    # 특정 특성 값 예측 (예: 12)
    try:
        feature_value_to_predict = 12
        predicted_target = model.predict(scaler.transform([[feature_value_to_predict]]))[0]
        print(f"{feature_column} = {feature_value_to_predict}에 대한 {target_column} 예측 값: {predicted_target:.2f}")
    except ValueError:
        print("유효한 정수를 입력하세요.")

# 실행 예시
# analyze_data('파일_경로.csv', '특성_열_이름', '목표_열_이름')
analyze_data('Data/한국가스공사_시간별 공급량_20181231.csv', feature_column='시간', target_column='공급량')
