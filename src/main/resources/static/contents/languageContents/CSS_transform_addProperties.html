카테고리 속성 속성값 속성 설명 사용 이유
전환(Transform) transform 값 (예: translate(10px, 20px), rotate(45deg)) 요소의 변형을 적용합니다. 요소의 위치, 크기, 회전 등을 조정하기 위해 사용합니다.
transform-origin 값 (예: top left, 50% 50%) 변형의 기준점을 설정합니다. 변형이 어떻게 적용될지를 조정하기 위해 사용합니다.
scale 값 (예: scale(1.5)) 요소의 크기를 조절합니다. 요소를 확대하거나 축소하기 위해 사용합니다.
rotate 값 (예: rotate(90deg)) 요소를 회전합니다. 요소의 방향을 변경하기 위해 사용합니다.
translate 값 (예: translate(10px, 20px)) 요소를 이동합니다. 요소의 위치를 조정하기 위해 사용합니다.
skew 값 (예: skew(10deg, 20deg)) 요소를 기울입니다. 요소의 형태를 변형하여 시각적 효과를 주기 위해 사용합니다.
추가 속성 align-content 값 (예: flex-start, center, space-between) 플렉스 또는 그리드 컨테이너의 여백을 설정합니다. 컨테이너 내 아이템의 배치를 조정하기 위해 사용합니다.
align-self 값 (예: auto, flex-start, center) 개별 플렉스 또는 그리드 아이템의 정렬을 설정합니다. 특정 아이템의 위치를 다른 아이템과 다르게 조정하기 위해 사용합니다.
grid-area 값 (예: header, main, footer) 그리드 아이템의 영역을 지정합니다. 아이템의 위치와 크기를 쉽게 조정하기 위해 사용합니다.
grid-template-areas 값 (예: "header header" "sidebar main") 그리드의 레이아웃을 정의합니다. 레이아웃을 시각적으로 이해하기 쉽게 만들기 위해 사용합니다.
