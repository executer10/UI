변수 (Variables)

제목 정의

  • 변수는 데이터를 저장하는 공간입니다.
  • 변수는 프로그래밍에서 중요한 역할을 합니다.

세부 정보 정의

  1. 변수 선언: 변수를 사용하기 전에 이름과 데이터 타입을 정합니다.
  2. 값 저장: 선언된 변수에 값을 할당합니다.
  3. 변수 사용: 저장된 값을 참조하여 사용합니다.
  4. 변수 사용 범위: 변수가 유효한 영역을 의미합니다. 변수가 선언된 위치에 따라 다릅니다.

특징

  1. 변수는 이름을 통해 쉽게 식별할 수 있습니다.
  2. 변수의 값은 필요에 따라 변경할 수 있습니다.
  3. 변수는 프로그램의 메모리에서 공간을 차지합니다.

사용 이유

  1. 데이터 관리: 프로그램 내 데이터를 효율적으로 관리하기 위해서입니다.
  2. 코드 가독성: 변수를 사용하여 코드를 더 이해하기 쉽게 만듭니다.
  3. 연산 수행: 변수를 통해 수학적 연산을 수행할 수 있습니다.
  4. 동적 처리: 변수를 사용하여 사용자 입력에 따라 동적으로 값을 변경할 수 있습니다.

예제 구문

// 변수 선언
int number; // 정수형 변수 선언
// 값 저장
number = 10; // 변수에 값 저장
// 변수 사용
System.out.println(number); // 변수의 값 출력

기본 타입

제목 정의

  • 기본 타입은 프로그래밍에서 데이터를 저장하는 기본적인 형식입니다.
  • 주로 정수 타입, 실수 타입, 논리 타입으로 나누어집니다.

세부 정보 정의

  1. 정수 타입: 정수 값을 저장합니다. 예: 1, 2, -3
  2. 실수 타입: 소수점을 포함한 값을 저장합니다. 예: 1.5, -2.3
  3. 논리 타입: 참(true) 또는 거짓(false) 값을 저장합니다.

특징

  1. 각 기본 타입은 특정한 메모리 크기를 차지합니다.
  2. 타입에 따라 저장할 수 있는 값의 범위가 다릅니다.
  3. 타입에 맞게 변수를 선언하여 데이터를 저장합니다.

사용 이유

  1. 효율적인 데이터 관리를 위해 필요합니다.
  2. 프로그램의 성능을 최적화할 수 있습니다.
  3. 각 타입에 맞는 연산을 수행하기 위해 사용됩니다.

예제 구문

// 정수 타입 변수 선언
int age = 25; // 나이를 저장
// 실수 타입 변수 선언
double height = 1.75; // 키를 저장
// 논리 타입 변수 선언
boolean isStudent = true; // 학생 여부 저장

기본 타입에는 다음과 같은 것들이 있습니다.

  • 문자형: char
  • 논리형: boolean

참조 타입(Reference Types)
참조 타입은 객체의 주소를 저장하며, 다음과 같은 것들이 있습니다:
  • 클래스(Class)
  • 인터페이스(Interface)
  • 배열(Array)
종류 설명
클래스(Class)
  • 객체를 생성하기 위한 탬플릿
  • 필드(데이터)와 메소드(행위)를 포함
  • 예: String name = new String("name");
    ArrayList<String> a = new ArrayList<>();
    사용자 정의 클래스 등
인터페이스(Interface)
  • 메소드의 선언만을 포함하는 추상 타입
  • 다중 상속을 구현하는 방법을 제공
  • 예: List, Runnable, Comparable 등
배열(Array)
  • 같은 타입의 여러 변수를 하나의 이름으로 그룹화
  • 인덱스를 통해 개별 요소에 접근
  • 예: int[], String[], Object[] 등

참조 타입의 특징
메모리 사용
  • 32비트 JVM: 4byte
  • 64비트 JVM: 8byte
null 값 참조 타입 변수는 null 값을 가질 수 있음 (어떤 객체도 참조하지 않음을 의미)
가비지 컬렉션 더 이상 참조되지 않는 객체는 자동으로 메모리에서 제거됨
다형성 상위 클래스 타입의 변수로 하위 클래스의 객체를 참조할 수 있음
