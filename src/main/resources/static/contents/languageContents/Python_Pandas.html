      • 구조와 특성

    • 시리즈

      • 생성 방법
  • 데이터 조작
    • 데이터 탐색
    • 데이터 전처리
    • 데이터 변환

  • 파일 처리

세부 내용

  1. 데이터 구조
    1. 데이터프레임 (DataFrame)
      • 정의: 2차원 레이블링된 데이터 구조
      • 구성 요소:
        • 인덱스(Index): 행 식별자
        • 컬럼(Columns): 열 식별자
        • 데이터(Data): 실제 저장된 값
      • 생성 방법:
        • 딕셔너리로 생성: pd.DataFrame(dict)
        • 리스트로 생성: pd.DataFrame(list)
        • 파일에서 생성: pd.read_csv(), pd.read_excel()
    2. 시리즈 (Series)
      • 정의: 1차원 레이블링된 배열
      • 구성 요소:
        • 인덱스(Index): 데이터 식별자
        • 데이터(Data): 실제 저장된 값
  2. 데이터 조작
    1. 데이터 탐색
      • 기본 정보 확인
        • head(): 상위 n개 행 확인
        • tail(): 하위 n개 행 확인
        • info(): 데이터프레임 정보 확인
        • describe(): 기초 통계량 확인
    2. 데이터 전처리
      • 결측치 처리
        • isna(): 결측치 확인
        • fillna(): 결측치 채우기
        • dropna(): 결측치 제거
      • 중복 처리
        • duplicated(): 중복 확인
        • drop_duplicates(): 중복 제거

특징

  1. 직관적인 데이터 구조와 조작 방법 제공
  2. 고성능 데이터 처리 및 분석 기능 지원
  3. 다양한 파일 형식과의 호환성
  4. 강력한 시계열 데이터 처리 기능
  5. SQL과 유사한 데이터 조작 문법
  6. 메모리 최적화된 데이터 처리
  7. 유연한 데이터 병합 및 결합 기능
  8. 풍부한 내장 분석 도구 제공

사용 이유

  1. 대용량 데이터 처리
    • 메모리 최적화된 데이터 구조로 대규모 데이터셋 처리 가능
    • 청크 단위 처리를 통한 효율적인 대용량 파일 처리
  2. 데이터 분석 및 전처리
    • 간편한 결측치 및 이상치 처리
    • 다양한 통계 분석 도구 제공
  3. 데이터 통합
    • 다양한 소스의 데이터를 쉽게 통합
    • 유연한 데이터 형식 변환
  4. 시계열 분석
    • 날짜/시간 데이터의 효율적 처리
    • 시계열 특화 분석 도구 제공
