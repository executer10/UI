Java

Java란?

 Java는 1995년 Sun MicroSystems에서 개발한 객체지향 프로그래밍 언어입니다.
Java의 주요 특징
1. 객체지향 언어 Java는 객체지향 프로그래밍(OOP) 패러다임을 따르는 언어로, 캡슐화, 상속, 다형성 등의 OOP개념을 완벽하게 지원합니다. 이를 통해 코드의 재사용성과 유지보수성을 높일 수 있습니다.
2. 플랫폼 독립성 Java는 JVM(Java Virtual Machine) 위에서 동작하기 때문에 운영체제에 상관없이 실행될 수 있습니다. 한번 작성된 자바 코드는 JVM이 설치된 모든 환경에서 별도의 수정 없이 실행 가능합니다.
3. 풍부한 API와 라이브러리 Java는 개발에 필요한 다양한 기능을 제공하는 방대한 API와 라이브러리를 가지고 있습니다. 이를 활용하면 네트워크 프로그래밍, 데이터베이스 연결, GUI개발 등을 쉽고 효율적으로 할 수 있습니다.
4. 메모리 관리 Java는 개발자가 직접 메모리를 관리할 필요가 없습니다. Garbage Collection 기능을 통해 사용하지 않는 객체를 자동으로 제거하여 메모리 누수를 방지합니다.
5. 멀티스레딩 지원 자바는 스레드 생성과 관리를 위한 간단하고 강력한 기능을 제공합니다. 멀티스레딩을 통해 프러그램의 반응성과 성능을 향상시킬 수 있습니다.
6. 자바의 활용 분야 Java는 다양한 분양에서 폭넓게 사용되는 언어입니다.
  • 웹 애플리케이션 개발: 서블릿, JSP, Spring framework 등을 활용한 서버 사이드 개발
  • 모바일 앱 개발 : 안드로이드 앱 개발의 주요 언어
  • 데스크톱 애플리케이션 개발
  • 임베디드 시스템 개발
  • 빅데이터 처리 : 하둡, 스파크 등의 framework에서 사용
  • 인공지능 및 머신러닝 : 딥러닝 라이브러리인 DL4J 등에서 활용
