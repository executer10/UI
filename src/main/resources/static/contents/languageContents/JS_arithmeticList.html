산술 연산자 설명 수식 수식의 값
+ 덧셈 x=3+2 5
- 뺄셈 x=3-2 1
* 곱셈 x=3*2 6
/ 나눗셈 x=3/2 1.5
% 나머지 x=3%2 1
++ 증가 ++x (혹은 x++) x의 값 3→4
-- 감소 --x (혹은 x--) x의 값 3→2
할당 연산자
(x=10, y=5) 		수식 동일한 수식 결과
+= x += y x = x + y x = 15
-= x -= y x = x - y x = 5
*= x *= y x = x * y x = 50
/= x /= y x = x / y x = 2
%= x %= y x = x % y x = 0
비교 연산자
(x=1) 		설명 수식 수식의 값
== 값이 같으면 참 x==1 true
x==2 false
!= 값이 다르면 참 x!=2 true
x!=1 false
> 크면 참 x>2 false
< 작으면 참 x<2 true
>= 크거나 같으면 참 x>=2 false
<= 작거나 같으면 참 x<=2 true
=== 값과 타입이 모두 같으면 참 x===1 true(값도 같고 타입도 같다.)
x==="1" false(값은 같으나 타입이 다르다.)
!== 값과 타입이 다르면 참 x!==1 false(값도 같고 타입도 같다.)
x==="1" true(값은 같으나 타입이 다르다.)
논리 연산자 기호 사용 방법 의미
&& x && y AND 연산. x와 y가 모두 참이면 true, 그렇지 않으면 false
|| x || y OR 연산. x나 y중에 하나라도 참이면 true, 모두 거짓이면 false
! !x NOT연산. x가 참이면 flase, x가 거짓이면 true
조건 연산자 max_value = (x>y) ? x : y; x가 y보다 크면 수식의 결과는 x가 되고, 그렇지 않으면 y
