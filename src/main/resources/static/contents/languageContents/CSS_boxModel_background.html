카테고리 속성 속성값 속성 설명 사용 이유
기본 속성 width 값(예: 100px, 50%) 요소의 너비를 설정합니다. 레이아웃을 구성할 때 사용합니다.
height 값(예: 100px, 50%) 요소의 높이를 설정합니다. 레이아웃을 구성할 때 사용합니다.
border 값(예: 1px solid black) 요소의 테두리를 설정합니다. 시각적 구분을 위해 사용합니다.
margin 값(예: 10px, auto) 요소의 외부 여백을 설정합니다. 요소 간의 간격 조절에 사용합니다.
padding 값(예: 10px) 요소의 내부 여백을 설정합니다. 내용과 테두리 간의 간격을 조절하는 데 사용합니다.
box-sizing content-box, border-box 박스 크기 계산 방식을 지정합니다. 레이아웃의 예측 가능성을 높이기 위해 사용합니다.
배경 background-color 색상값 (예: red, #ff0000) 요소의 배경색을 설정합니다. 시각적 효과 및 구분을 위해 사용합니다.
background-image URL (예: url('image.jpg')) 요소의 배경 이미지를 설정합니다. 배경을 통해 디자인을 강화하기 위해 사용합니다.
background-size 값 (예: cover, contain) 배경 이미지의 크기를 설정합니다. 배경 이미지가 요소에 맞게 조정되도록 사용합니다.
background-repeat 값 (예: no-repeat, repeat) 배경 이미지의 반복 여부를 설정합니다. 배경 이미지의 표시 방식을 조절하기 위해 사용합니다.
background-position 값 (예: center, top left) 배경 이미지의 위치를 설정합니다. 배경 이미지의 시작점을 조정하기 위해 사용합니다.
opacity 값 (0.0 - 1.0) 요소의 투명도를 설정합니다. 시각적 효과를 위해 요소의 투명도를 조절하기 위해 사용합니다.
