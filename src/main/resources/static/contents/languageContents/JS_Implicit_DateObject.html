new Date()객체 메소드 설명 사용 이유
날짜 가져오기 메소드 getDay() 현지 시간 기준으로 지정된 날짜의 일(1-31)을 반환 특정 날짜의 일자만 필요할 때 사용. 예: 월별 보고서 생성, 일정 관리 등
getDay() 현지 시간 기준으로 지정된 날짜 요일(0-6)을 반환 요일 기반 로직 구현 시 사용. 예: 주말 확인, 영업일 계산 등
getFullYear() 현지 시간 기준으로 지정된 날짜의 연도(4자리)를 반환 연도 정보가 필요한 경우 사용. 예: 나이 계산, 연간 통계 등
getMonth() 현지 시간 기준으로 지정된 날짜의 월(0-11)을 반환 월 단위 작업 시 사용. 예: 월별 데이터 집계, 계절 판단 등
시간 가져오기 메소드 getHours() 현지 시간 기준으로 지정된 날짜의 시간(0-23)을 반환 시간대별 작업 수행 시 사용. 예: 업무 시간 확인, 시간대별 알림 등
getMinutes() 현지 시간 기준으로 지정된 날짜의 분(0-59)을 반환 정확한 시간 표시나 계산이 필요할 때 사용. 예: 경과 시간 계산, 스케줄링 등
getSeconds() 현지 시간 기준으로 지정된 날짜의 초(0-59)을 반환 초 단위의 정밀한 시간 측정이 필요할 때 사용. 예: 스톱워치 기능, 성능 측정 등
getMilliseconds() 현지 시간 기준으로 지정된 날짜의 밀리초(0-999)을 반환 매우 정밀한 시간 측정이 필요한 경우 사용. 예: 애니메이션 타이밍, 고성능 측정 등
날짜 설정 메소드 setDate() 현지 시간 기준으로 날짜 객체의 일을 설정 특정 날짜로 변경해야 할 때 사용. 예: 다음 달 같은 날짜 설정, 날짜 조정 등
setMonth() 현지 시간 기준으로 날짜 객체의 월을 설정 월 단위로 날짜를 조정해야 할 때 사용. 예: 월별 데이터 순회, 계절 변경 등
setFullYear() 현지 시간 기준으로 날짜 객체의 연도를 설정 연도를 변경해야 할 때 사용. 예: 미래/과거 날짜 설정, 연도별 데이터 처리 등
시간 설정 메소드 setHour() 현지 시간 기준으로 날짜 객체의 시간을 설정 특정 시간으로 변경해야할 때 사용. 예: 타이머 설정, 스케쥴 조정 등
setMinutes() 현지 시간 기준으로 날짜 객체의 분을 설정 분 단위로 시간을 조정해야 할 때 사용. 예: 타이머 설정, 스케줄 조정 등
setSeconds() 현지 시간 기준으로 날짜 객체의 초를 설정 초 단위의 정밀한 시간 설정이 필요할 때 사용. 예: 카운터 타이머, 정밀 시간 조정 등
setMilliseconds() 현지 시간 기준으로 날짜 객체의 밀리초를 설정 밀리초 단위의 매우 정밀한 시간 설정이 필요할 때 사용. 예: 고성능 타이밍, 애니메이션 제어 등
기타 유용한 메소드 toString() 날짜 객체를 문자열로 변환 날짜를 읽기 쉬운 문자열로 표시할 때 사용. 예: 로그 기록, 디버깅 등
toDateString() 날짜 객체의 날짜 부분을 사람이 읽을 수 있는 문자열로 반환 날짜만 문자열로 필요할 때 사용. 예: 간단한 날짜 표시, 날짜 비교 등
toLocaleString() 로케일 규칙을 사용하여 날짜 객체를 문자열로 변환 지역화된 날짜 형식이 필요할 때 사용. 예: 다국어 지원 애플리케이션, 지역별 날짜 표시 등
toISOString() ISO 표준을 사용하여 날짜를 문자열로 반환 표준화된 날짜 형식이 필요할 때 사용. 예: API 통신, 데이터베이스 저장 등
getTime() 1970년 1월 1일부터 경과된 밀리초 수를 반환 두 날짜 간의 시간 차이 계산 시 사용. 예: 경과 시간 계산, 날짜 비교 등
now() 1970년 1월 1일 이후 현재까지 경과한 밀리초를 반환 현재 시간을 밀리초로 얻고 싶을 때 사용. 예: 타임스탬프 생성, 성능 측정 등
parse() 날짜 문자열을 파싱하여 1970년 1월 1일부터 경과한 밀리초 수를 반환 문자열로 된 날짜를 Date 객체로 변환할 때 사용. 예: 사용자 입력 날짜 처리, 날짜 파싱 등
toJSON() JSON 날짜 형식의 문자열을 반환 날짜를 JSON 형식으로 직렬화할 때 사용. 예: API 응답 생성, 데이터 저장 등
valueOf() 날짜 객체의 원시값(1970년 1월 1일부터 경과한 밀리초)을 반환 날짜를 숫자로 표현해야 할 때 사용. 예: 날짜 정렬, 수학적 연산 등
