카테고리 속성 속성값 속성 설명 사용 이유
레이아웃 display 값 (예: block, inline, flex) 요소의 표시 방식을 지정합니다. 요소의 레이아웃 구조를 결정하기 위해 사용합니다.
position 값 (예: static, relative, absolute, fixed) 요소의 위치 지정 방법을 설정합니다. 요소의 배치 방식을 조절하기 위해 사용합니다.
float 값 (예: left, right, none) 요소를 좌우로 띄웁니다. 텍스트와 다른 요소들 사이의 배치를 조정하기 위해 사용합니다.
z-index 정수 값 요소의 쌓임 순서를 지정합니다. 겹치는 요소들 간의 우선순위를 설정하기 위해 사용합니다.
그리드 레이아웃 grid-template-columns 값 (예: 100px 1fr 2fr) 그리드의 열 구조를 정의합니다. 그리드의 열 비율과 크기를 설정하기 위해 사용합니다.
grid-template-rows 값 (예: 100px auto) 그리드의 행 구조를 정의합니다. 그리드의 행 비율과 크기를 설정하기 위해 사용합니다.
grid-column 값 (예: 1 / 3) 그리드 아이템의 열 위치를 지정합니다. 아이템의 위치를 조정하기 위해 사용합니다.
grid-row 값 (예: 2 / 4) 그리드 아이템의 행 위치를 지정합니다. 아이템의 위치를 조정하기 위해 사용합니다.
gap 값 (예: 10px) 그리드 셀 간의 간격을 설정합니다. 셀 간의 간격을 조정하여 레이아웃을 개선하기 위해 사용합니다.
플렉스박스 flex-direction 값 (예: row, column) 플렉스 아이템의 배치 방향을 설정합니다. 아이템의 흐름 방향을 결정하기 위해 사용합니다.
justify-content 값 (예: flex-start, center, space-between) 주 축에서 아이템의 정렬 방식을 설정합니다. 아이템 간의 간격을 조정하기 위해 사용합니다.
align-items 값 (예: stretch, center, flex-start) 교차 축에서 아이템의 정렬 방식을 설정합니다. 아이템의 높이를 조정하여 레이아웃을 개선하기 위해 사용합니다.
flex-wrap 값 (예: nowrap, wrap) 플렉스 아이템의 줄 바꿈 여부를 설정합니다. 아이템이 공간 부족 시 처리 방식을 결정하기 위해 사용합니다.
flex-grow 정수 값 (예: 1, 2) 플렉스 아이템의 증가 비율을 설정합니다. 남은 공간을 어떻게 분배할지를 결정하기 위해 사용합니다.
flex-shrink 정수 값 (예: 1, 0) 플렉스 아이템의 축소 비율을 설정합니다. 공간이 부족할 때 아이템의 크기를 조정하기 위해 사용합니다.
