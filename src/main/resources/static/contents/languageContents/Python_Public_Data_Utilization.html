세부 내용

  1. 데이터 분석
    • 인구통계
      1. 총인구수와 출생아수 분석
        • pandas.read_csv(): 통계청 데이터 로드
        • pandas.DataFrame.groupby(): 연도별 데이터 그룹화
        • matplotlib.pyplot: 차트 생성 및 시각화
        • 데이터 전처리 및 결측치 처리 방법
      2. 지역별 인구현황 분석
        • pandas.read_csv(): 인구현황 데이터 로드
        • pandas.DataFrame.dropna(): 결측치 제거
        • pandas.DataFrame.loc[]: 특정 지역 데이터 필터링
        • matplotlib.pyplot: 지역별 인구 차트 생성
  2. 데이터 시각화
    • 지리정보
      1. 공원정보 지도 표현
        • folium.Map(): 기본 지도 생성
        • folium.Marker(): 위치 마커 추가
        • folium.Map.add_child(): 마커 및 레이어 추가
        • 위도/경도 좌표 시스템 활용

특징

  1. 실시간 데이터 분석: 공공 API를 통한 최신 데이터 접근
  2. 다양한 시각화: 차트, 지도 등 다중 시각화 도구 활용
  3. 데이터 전처리: 결측치 처리, 데이터 정제 기능 포함
  4. 사용자 상호작용: 지역 선택, 기간 설정 등 동적 분석 지원

사용 이유

  1. 정책 의사결정: 인구 통계 기반의 정책 수립 지원
  2. 자원 최적화: 공공시설 위치 선정 및 자원 배분 최적화
  3. 트렌드 분석: 인구 변화 추이 및 지역별 특성 파악
  4. 시각적 소통: 복잡한 데이터의 직관적 전달 가능

예제 구문

# 1. 인구 데이터 분석 예제
import pandas as pd
import matplotlib.pyplot as plt

# 데이터 로드
population_data = pd.read_csv('population.csv', encoding='utf-8')

# 연도별 총인구수 분석
yearly_population = population_data.groupby('year')['total_population'].sum()

# 시각화
plt.figure(figsize=(10, 6))
plt.plot(yearly_population.index, yearly_population.values, marker='o')
plt.title('연도별 총인구수 추이')
plt.xlabel('연도')
plt.ylabel('총인구수')
plt.grid(True)
plt.show()

# 2. 공원정보 지도 시각화 예제
import folium

# 서울시 중심 지도 생성
seoul_map = folium.Map(
    location=[37.5665, 126.9780],
    zoom_start=12
)

# 공원 정보 추가
parks_data = [
    {"name": "서울숲", "location": [37.5447, 127.0374]},
    {"name": "올림픽공원", "location": [37.5202, 127.1214]}
]

for park in parks_data:
    folium.Marker(
        location=park["location"],
        popup=park["name"],
        icon=folium.Icon(color='green')
    ).add_to(seoul_map)

# 지도 저장
seoul_map.save('seoul_parks.html')

# 3. 지역별 인구현황 분석 예제
# 특정 지역 데이터 필터링
seoul_data = population_data.loc[population_data['region'] == 'Seoul']

# 결측치 제거
seoul_data_clean = seoul_data.dropna()

# 연령대별 인구 분포 시각화
plt.figure(figsize=(12, 6))
seoul_data_clean.plot(
    kind='bar',
    x='age_group',
    y='population',
    title='서울시 연령대별 인구 분포'
)
plt.xticks(rotation=45)
plt.tight_layout()
plt.show()
