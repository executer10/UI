카테고리 속성 속성값 속성 설명 사용 이유
트랜지션 transition 값 (예: all 0.3s ease) CSS 속성 변화를 부드럽게 처리합니다. 변화 효과를 부드럽게 만들어 사용자 경험을 향상시키기 위해 사용합니다.
transition-property 속성명 (예: opacity, transform) 트랜지션을 적용할 CSS 속성을 지정합니다. 어떤 속성에 트랜지션을 적용할지를 명확히 하기 위해 사용합니다.
transition-duration 시간 (예: 0.5s, 300ms) 트랜지션의 지속 시간을 설정합니다. 변화가 얼마나 오래 지속될지를 조절하기 위해 사용합니다.
transition-timing-function 값 (예: linear, ease-in, ease-out) 트랜지션의 속도 곡선을 지정합니다. 변화의 속도를 조절하여 자연스러운 느낌을 주기 위해 사용합니다.
애니메이션 animation 값 (예: slide 1s ease-in-out) 키프레임 애니메이션을 적용합니다. 요소에 애니메이션 효과를 부여하여 시각적 흥미를 주기 위해 사용합니다.
@keyframes 이름 (예: slide) 애니메이션의 중간 상태를 정의합니다. 애니메이션의 세부 동작을 설정하기 위해 사용합니다.
animation-name 이름 (예: slide) 애니메이션의 이름을 지정합니다. 어떤 애니메이션을 사용할지를 명확히 하기 위해 사용합니다.
animation-duration 시간 (예: 2s, 500ms) 애니메이션의 지속 시간을 설정합니다. 애니메이션의 속도를 조절하기 위해 사용합니다.
animation-delay 시간 (예: 1s, 300ms) 애니메이션 시작 전 대기 시간을 설정합니다. 애니메이션 시작 시점을 조절하기 위해 사용합니다.
효과 box-shadow 값 (예: 2px 2px 5px rgba(0, 0, 0, 0.5)) 요소에 그림자 효과를 추가합니다. 요소를 강조하고 입체감을 주기 위해 사용합니다.
text-shadow 값 (예: 1px 1px 2px rgba(0, 0, 0, 0.5)) 텍스트에 그림자 효과를 추가합니다. 텍스트를 강조하고 가독성을 높이기 위해 사용합니다.
filter 값 (예: blur(5px), grayscale(100%)) 요소에 시각적 효과를 적용합니다. 디자인 효과를 추가하고 시각적 흥미를 유도하기 위해 사용합니다.
transition-delay 시간 (예: 0.5s, 200ms) 트랜지션 시작 전 대기 시간을 설정합니다. 트랜지션의 시작 시점을 조절하기 위해 사용합니다.
