JavaScript
JavaScript란? JavaScript는 웹브라우저에서 동작하는 스크립트 언어로, 웹페이지를 동저깅고 상호작용하게 만드는 데 사용합니다.
1. 웹 브라우저에서 실행 JavaScript는 웹 브라우저에서 실행되는 클라이언트로 사이드 스크립트 언어입니다. 웹 페이지에 역동성과 상호작용을 더하기 위해 사용되며, 사용자의 이벤트에 반응하여 실시간으로 콘텐츠를 변경할 수 있습니다.
2. 객체 기반 언어 JavaScript는 객체 기반 언어로 객체를 사용하여 데이터와 기능을 캡슐화 합니다. 함수, 배열, 날짜 등 다양한 내장 객체를 제공하며, 사용자 정의 객체도 생성할 수 있습니다.
3. 동적 타입 언어 JavaScript는 변수의 데이터 타입을 미리 선언할 필요가 없는 동적 타입 언어 입니다. 변수에 어떤 타입의 값도 할당할 수 있으며, 자동으로 타입이 결정됩니다.
4. 함수형 프로그래밍 지원 JavaScript는 일급 함수를 지원하여 함수형 프로그래밍 패러다임을 따를 수 있습니다. 함수를 변수에 할당하거나 다른 함수의 인자로 전달할 수 있으며, 함수를 반환값으로 사용할 수도 있습니다.
5. 비동기 처리 JavaScript는 비동기 처리를 지원하여 블로킹 없이 여러 작업을 동시에 수행할 수 있습니다. 콜백 함수, Promise, async/await 등의 기능을 통해 비동기 코드를 더 쉽게 작성할 수 있습니다.
JavaScript의 활용 분야 JavaScript는 웹 개발에서 폭넓게 활용되고 있습니다.
  • 웹 페이지 동적 기능 구현: 폼 유효성 검사, 애니메이션, 콘텐츠 동적 변경 등
  • 웹 애플리케이션 개발: 싱글 페이지 애플리케이션(SPA) 개발 등
  • 서버 사이드 개발: Node.js를 사용한 서버 개발
  • 모바일 앱 개발: React Native, lonic 등의 framework를 활용한 크로스 플랫폼 앱 개발
  • 게임 개발: HTML5와 함께 사용하여 웹 기반 게임 개발
  • 프레젠테이션과 슬라이드쇼 제작: Reveal.js, Bespoke.js 등의 framework 활용
