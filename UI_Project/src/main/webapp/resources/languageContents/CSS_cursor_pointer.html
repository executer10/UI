카테고리 속성 속성값 속성 설명 사용 이유
커서 및 포인터 cursor 값 (예: pointer, default, text) 마우스 커서의 모양을 설정합니다. 사용자 인터페이스에서 클릭 가능한 요소를 시각적으로 나타내기 위해 사용합니다.
pointer-events 값 (예: auto, none) 요소에 대한 포인터 이벤트를 설정합니다. 특정 요소가 클릭이나 터치 이벤트를 수신할지를 제어하기 위해 사용합니다.
touch-action 값 (예: manipulation, none) 터치 동작의 기본 동작을 설정합니다. 터치 이벤트의 동작을 조정하여 사용자 경험을 개선하기 위해 사용합니다.
