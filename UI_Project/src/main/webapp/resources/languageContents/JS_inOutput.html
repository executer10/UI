함수명(매개변수) 설명 사용 이유 예제 코드
console.log(value) 콘솔에 값을 출력 디버깅 및 로그를 확인하기 위해 사용 console.log("Hello, world!");
alert(message) 경고 대화상자를 띄우고 메세지를 표시 사용자에게 정보를 간단히 알릴 때 유용함 alert("안녕하세요!")
prompt(message) 사용자에게 입력을 요청하는 대화상자를 띄움 사용자로부터 정보를 입력받을 때 사용됨 let name = prompt(`이름을 입력하세요: `);
confirm(message) 확인 또는 취소를 요구하는 대화상자를 띄움 사용자의 선택을 확인할 때 유용함 let result = confirm(`정말로 삭제하시겠습니까?`);
document.write(content) 문서에 내용을 직접 작성함 간단한 웹 페이지를 만들 때 유용하지만, 기존의 페이지를 초기화후 다시 입력하는 형태. document.write(`<h1>Hellow, World!</h1>`);
fetch(url) 네트워크 요청을 통해 데이터를 가져옴 API와 통신하거나 외부 리소스를 가져올 때 사용됨 fetch("https://api.example.com/data").then(response => response.japi.example.com/data").then(response => response.json()).then(data => console.log(data));
