카테고리 속성 속성값 속성 설명 사용 이유
반응형 디자인 @media 미디어 쿼리 조건 (예: @media (max-width: 600px)) 미디어 쿼리를 사용한 반응형 스타일을 적용합니다. 화면 크기에 따라 스타일을 조정하여 다양한 디바이스에 최적화하기 위해 사용합니다.
max-width 값 (예: 1200px) 요소의 최대 너비를 설정합니다. 요소가 너무 넓어지지 않도록 제한하기 위해 사용합니다.
min-width 값 (예: 300px) 요소의 최소 너비를 설정합니다. 요소가 너무 좁아지지 않도록 보장하기 위해 사용합니다.
테이블 관련 속성 border-collapse 값 (예: collapse, separate) 테이블 경계 처리 방식을 설정합니다. 테이블의 경계를 더 깔끔하게 보이도록 만들기 위해 사용합니다.
border-spacing 값 (예: 5px) 테이블 셀 간의 간격을 설정합니다. 셀 간의 여백을 조정하여 가독성을 높이기 위해 사용합니다.
caption-side 값 (예: top, bottom) 테이블 캡션의 위치를 설정합니다. 캡션의 가시성을 높이기 위해 사용합니다.
empty-cells 값 (예: show, hide) 빈 셀의 표시 여부를 설정합니다. 빈 셀을 표시하거나 숨겨서 테이블을 깔끔하게 만들기 위해 사용합니다.
table-layout 값 (예: auto, fixed) 테이블의 레이아웃 방식을 설정합니다. 테이블의 열 너비 계산 방식을 조정하기 위해 사용합니다.
