카테고리 속성 속성값 속성 설명 사용 이유
텍스트 스타일링 font-family 값 (예: Arial, 'Times New Roman') 텍스트의 글꼴을 지정합니다. 디자인과 가독성을 위해 사용합니다.
font-size 값 (예: 16px, 1.5em) 글자 크기를 설정합니다. 텍스트의 가독성을 높이기 위해 사용합니다.
font-weight 값 (예: normal, bold, 700) 글자 굵기를 설정합니다. 중요한 텍스트를 강조하기 위해 사용합니다.
font-style 값 (예: normal, italic, oblique) 텍스트의 글꼴 스타일을 지정합니다. 텍스트에 다양한 스타일 효과를 주기 위해 사용합니다.
color 색상값 (예: red, #ff0000) 텍스트의 색상을 설정합니다. 시각적 효과를 높이기 위해 사용합니다.
text-align 값 (예: left, center, right) 텍스트의 수평 정렬 방식을 지정합니다. 레이아웃의 균형을 맞추기 위해 사용합니다.
line-height 값 (예: 1.5, 20px) 텍스트의 줄 간격을 설정합니다. 가독성을 높이고 텍스트의 밀도를 조절하기 위해 사용합니다.
text-decoration 값 (예: none, underline, line-through) 텍스트에 장식을 설정합니다. 특정 텍스트를 강조하거나 스타일을 추가하기 위해 사용합니다.
word-break 값 (예: normal, break-all, keep-all) 단어가 줄 바꿈을 할 때의 처리 방식을 지정합니다. 긴 단어가 줄을 넘어갈 때의 가독성을 조절하기 위해 사용합니다.
