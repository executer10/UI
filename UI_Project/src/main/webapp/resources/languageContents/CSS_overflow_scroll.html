카테고리 속성 속성값 속성 설명 사용 이유
오버플로우 및 스크롤 overflow 값 (예: visible, hidden, scroll, auto) 요소의 내용이 넘칠 때 처리 방법을 설정합니다. 내용이 넘칠 경우의 표시 방식을 조정하기 위해 사용합니다.
overflow-x 값 (예: visible, hidden, scroll, auto) 수평 방향의 오버플로우 처리 방법을 설정합니다. 수평 내용이 넘칠 때의 표시 방식을 조정하기 위해 사용합니다.
overflow-y 값 (예: visible, hidden, scroll, auto) 수직 방향의 오버플로우 처리 방법을 설정합니다. 수직 내용이 넘칠 때의 표시 방식을 조정하기 위해 사용합니다.
scroll-behavior 값 (예: auto, smooth) 스크롤 동작의 부드러움을 설정합니다. 사용자 경험을 향상시키기 위해 스크롤을 부드럽게 만들기 위해 사용합니다.
