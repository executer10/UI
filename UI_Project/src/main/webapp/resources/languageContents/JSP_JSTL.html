세부 내용

  1. JSTL 개요

    JSTL(JSP Standard Tag Library)은 JSP 페이지에서 자주 사용되는 핵심 기능(반복문, 조건문, 데이터 관리 등)을 구현한 커스텀 태그 라이브러리입니다. JSP 코드를 더 간단하고 가독성 있게 작성할 수 있도록 도와줍니다.

  2. JSTL 태그 라이브러리 종류
    라이브러리 접두어 기능 URI(통캣 8~9버전 기준)
    Core c 변수지원, 제어문, 페이지 관련 처리 http://java.sun.com/jsp/jstl/core
    XML x XML 관련 처리 http://java.sun.com/jsp/jstl/xml
    Formatting fmt 숫자/날짜/시간 포맷지정, 국제화 지원 http://java.sun.com/jsp/jstl/fmt
    Database sql DB 관련 CRUD http://java.sun.com/jsp/jstl/sql
    Functions fn 컬렉션, 문자열 처리 http://java.sun.com/jsp/jstl/functions
  3. JSTL 설정 방법
    1. Maven 의존성 추가 
      <dependency>
    <groupId>javax.servlet</groupId>
    <artifactId>jstl</artifactId>
    <version>1.2</version>
</dependency>
      혹은 mvnrepository.com에 접속하여 JSTL을 다운받아 WEB-INF의 lib폴더에 설치
    2. JSP 피이지에 태그 라이브러리 선언 
      <%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>
  4. 각 태그별 기능
    1. Core 태그 (c:)
      • set: 변수 선언 및 값 설정
      • remove: 선언된 변수 제거
      • out: 값을 출력
      • if: 조건문 처리
      • c:choose: 조건부 실행 태그
      • c:when: choose의 하위 태그, if 혹은 else if
      • c:otherwise: when 조건이 false일 때 실행
      • forEach: 컬렉션/배열의 반복 처리
      • forTokens: 구분자로 분리된 토큰 처리
      • import: 외부 자원 가져오기
      • url: URL 생성
      • param: URL 파라미터 추가
      • redirect: 페이지 리다이렉트
      • catch: 예외 처리
    2. XML 태그
      • out: XPath 표현식 결과 출력
      • parse: XML 데이터 파싱
      • parseError: XML 파싱 에러 처리
      • set: XPath 표현식 값으로 변수 설정
      • if: XML 조건문 처리
      • forEach: XML 노드 반복 처리
      • transform: XSL 변환 적용
      • param: XSLT 스타일시트 파라미터 설정
      • choose: XML 다중 조건문
      • when: XML 조건 분기
      • otherwise: XML 기본 조건
    3. Formatting 태그
      • formatNumber: 숫자 포맷팅
      • formatDate: 날짜/시간 포맷팅
      • parseNumber: 문자열을 숫자로 파싱
      • parseDate: 문자열을 날짜로 파싱
      • timeZone: 시간대 설정
      • setTimeZone: 시간대 변경
      • bundle: 리소스 번들 로드
      • setBundle: 리소스 번들 설정
      • setLocale: 로케일 설정
      • message: 메시지 출력
      • requestEncoding: 요청 인코딩 설정
    4. Database 태그
      • setDataSource: 데이터베이스 연결 설정
      • query: SELECT 쿼리 실행
      • update: INSERT/UPDATE/DELETE 쿼리 실행
      • param: SQL 파라미터 설정
      • dateParam: 날짜 타입 파라미터 설정
      • transaction: 트랜잭션 처리
      • setTimeZone: DB 시간대 설정
      • driver: JDBC 드라이버 설정
      • resultSet: 결과셋 처리
      • statement: SQL 문 실행
    5. Functions 태그
      • contains: 문자열 포함 여부 확인
      • containsIgnoreCase: 대소문자 무시 문자열 포함 확인
      • endsWith: 지정된 문자열로 끝나는지 확인
      • escapeXml: XML 마크업 문자 이스케이프 처리
      • indexOf: 문자열 위치 검색
      • join: 배열 요소를 문자열로 결합
      • length: 컬렉션/문자열 길이 반환
      • replace: 문자열 치환
      • split: 문자열 분할
      • startsWith: 지정된 문자열로 시작하는지 확인
      • substring: 부분 문자열 추출
      • substringAfter: 지정된 문자열 이후 부분 추출
      • substringBefore: 지정된 문자열 이전 부분 추출
      • toLowerCase: 소문자 변환
      • toUpperCase: 대문자 변환
      • trim: 문자열 앞뒤 공백 제거
  5. 각 태그별 속성
    • Core 태그 속성
      1. set 태그
        • var: 변수명
        • value: 설정할 값
        • target: 대상 객체
        • property: 프로퍼티명
        • scope: 변수 범위 (page, request, session, application)
      2. out 태그
        • value: 출력할 값
        • default: 기본값
        • escapeXml: XML 특수문자 이스케이프 여부
      3. if 태그
        • test: 조건식
        • var: 조건 결과 저장 변수
        • scope: 변수 범위
      4. choose 태그
        • 없음
      5. when 태그
        • test : 조건식
      6. otherwise 태그
        • 없음
      7. forEach 태그
        • var: 현재 항목 변수명
        • items: 반복할 컬렉션
        • begin: 시작 인덱스
        • end: 종료 인덱스
        • step: 증가값
        • varStatus: 반복 상태 변수
      8. forTokens 태그
        • var: 현재 토큰 변수
        • items: 문자열
        • delims: 구분자
        • varStatus: 반복 상태 변수
      9. import 태그
        • url: 가져올 리소스 URL
        • var: 저장할 변수명
        • scope: 변수 범위
        • charEncoding: 문자 인코딩
      10. url 태그
        • value: URL 경로
        • var: 저장할 변수명
        • scope: 변수 범위
      11. redirect 태그
        • url: 리다이렉트할 URL
        • context: 컨텍스트 경로
        • var: 저장할 변수명
        • scope: 변수 범위
      12. catch 태그
        • var: 예외 저장 변수
        • scope: 변수 범위
      13. param 태그
        • name: 파라미터 이름
        • value: 파라미터 값
      14. contains 태그
        • string: 검색 대상 문자열
        • substring: 찾을 부분 문자열
      15. containsIgnoreCase 태그
        • string: 검색 대상 문자열
        • substring: 찾을 부분 문자열
      16. length 태그
        • string: 길이를 측정할 문자열 또는 컬렉션
    • Formatting 태그 속성
      1. formatNumber 태그
        • value: 포맷팅할 숫자
        • type: 포맷 유형 (number, currency, percent)
        • pattern: 사용자 정의 패턴
        • currencyCode: 통화 코드
        • groupingUsed: 자릿수 구분 사용 여부
        • maxIntegerDigits: 최대 정수 자릿수
        • minIntegerDigits: 최소 정수 자릿수
        • maxFractionDigits: 최대 소수점 자릿수
        • minFractionDigits: 최소 소수점 자릿수
      2. formatDate 태그
        • value: 포맷팅할 날짜
        • type: 포맷 유형 (date, time, both)
        • dateStyle: 날짜 스타일 (default, short, medium, long, full)
        • timeStyle: 시간 스타일 (default, short, medium, long, full)
        • pattern: 사용자 정의 패턴
        • timeZone: 시간대
      3. parseNumber 태그
        • value: 파싱할 문자열
        • type: 파싱 유형 (number, currency, percent)
        • pattern: 사용자 정의 패턴
        • parseLocale: 파싱할 로케일
        • integerOnly: 정수만 파싱 여부
      4. parseDate 태그
        • value: 파싱할 문자열
        • type: 파싱 유형 (date, time, both)
        • dateStyle: 날짜 스타일
        • timeStyle: 시간 스타일
        • pattern: 사용자 정의 패턴
        • timeZone: 시간대
        • parseLocale: 파싱할 로케일
      5. bundle/setBundle 태그
        • basename: 리소스 번들 기본 이름
        • var: 저장할 변수명
        • scope: 변수 범위
      6. message 태그
        • key: 메시지 키
        • bundle: 리소스 번들
        • var: 저장할 변수명
        • scope: 변수 범위
      7. timeZone/setTimeZone 태그
        • value: 시간대 ID
        • var: 저장할 변수명
        • scope: 변수 범위
      8. setLocale 태그
        • value: 로케일 값
        • variant: 로케일 변형
        • scope: 변수 범위
      9. requestEncoding 태그
        • value: 인코딩 값
    • Database 태그 속성
      1. setDataSource 태그
        • driver: JDBC 드라이버 클래스
        • url: 데이터베이스 URL
        • user: 사용자명
        • password: 비밀번호
        • dataSource: 데이터소스 객체
        • var: 변수명
        • scope: 변수 범위
      2. query 태그
        • sql: SQL 쿼리문
        • var: 결과 저장 변수
        • scope: 변수 범위
        • dataSource: 데이터소스 참조
        • maxRows: 최대 행 수
      3. update 태그
        • sql: SQL 쿼리문
        • dataSource: 데이터소스 참조
        • var: 결과 저장 변수
        • scope: 변수 범위
      4. transaction 태그
        • dataSource: 데이터소스 참조
        • isolation: 트랜잭션 격리 수준
      5. param 태그
        • value: 파라미터 값
        • type: SQL 타입
      6. dateParam 태그
        • value: 날짜 값
        • type: 날짜 타입
        • timestamp: 타임스탬프 여부
      7. resultSet 태그
        • var: 결과셋 저장 변수
        • scope: 변수 범위
        • maxRows: 최대 행 수
    • XML 태그 속성
      1. parse 태그
        • xml: XML 문서
        • var: 파싱 결과 저장 변수
        • scope: 변수 범위
        • varDom: DOM 트리 저장 변수
      2. transform 태그
        • xml: XML 문서
        • xslt: XSLT 스타일시트
        • var: 변환 결과 저장 변수
        • scope: 변수 범위
      3. out 태그
        • select: XPath 표현식
        • escapeXml: XML 이스케이프 처리 여부
      4. if 태그
        • select: XPath 조건식
        • var: 조건 결과 저장 변수
        • scope: 변수 범위
      5. forEach 태그
        • select: XPath 표현식
        • var: 현재 항목 변수
        • varStatus: 반복 상태 변수
      6. param 태그
        • name: 파라미터 이름
        • value: 파라미터 값
      7. choose/when/otherwise 태그
        • select: XPath 조건식 (when 태그에만 해당)

특징

  • 간편한 문법: JSP 스크립틀릿 대체
  • 표준화된 태그: 일관된 코딩 스타일 제공
  • 다양한 기능: 핵심 기능 내장
  • 높은 재사용성: 공통 기능 모듈화

예제 구문

  • Core 태그 예제 
    <%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>

<!-- 1. 변수 설정 (set) -->
<c:set var="사용자이름" value="홍길동" />
<c:set var="나이" value="25" />
<c:set var="취미목록" value="등산,독서,여행" />
<c:set var="회원정보" value="${memberDAO.getMemberInfo()}" /> <!-- 객체 저장 예시 -->


    

<!-- 2. 변수 출력 (out) -->
회원이름: <c:out value="${사용자이름}" default="이름없음" />
<!-- 실행결과: 회원이름: 홍길동 -->


    

<!-- 3. 조건문 (if) -->
<c:if test="${나이 >= 20}">
    <p>${사용자이름}님은 성인입니다.</p>
</c:if>
<!-- 실행결과: 홍길동님은 성인입니다. -->


    

<!-- 4. 다중 조건문 (choose, when, otherwise) -->
<c:choose>
    <c:when test="${나이 >= 20 && 나이 < 30}">
        <p>20대입니다.</p>
    </c:when>
    <c:when test="${나이 >= 30 && 나이 < 40}">
        <p>30대입니다.</p>
    </c:when>
    <c:otherwise>
        <p>기타 연령대입니다.</p>
    </c:otherwise>
</c:choose>
<!-- 실행결과: 20대입니다. -->


    

<!-- 5. 반복문 (forEach) - 숫자 범위 -->
<h4>카운트다운</h4>
<c:forEach var="숫자" begin="1" end="3">
    <p>${숫자}번째 항목</p>
</c:forEach>
<!-- 실행결과:
1번째 항목
2번째 항목
3번째 항목
-->


    

<!-- 6. 반복문 (forEach) - 배열/리스트 -->
<%
    List<String> 과일목록 = Arrays.asList("사과", "바나나", "오렌지");
    request.setAttribute("과일들", 과일목록);
%>
<h4>과일 목록</h4>
<c:forEach var="과일" items="${과일들}" varStatus="상태">
    ${상태.count}. ${과일}<br>
</c:forEach>
<!-- 실행결과:
1. 사과
2. 바나나
3. 오렌지
-->


    

<!-- 7. 구분자로 문자열 분리 (forTokens) -->
<h4>취미 목록</h4>
<c:forTokens var="취미" items="${취미목록}" delims=",">
    - ${취미}<br>
</c:forTokens>
<!-- 실행결과:
- 등산
- 독서
- 여행
-->


    

<!-- 8. 예외 처리 (catch) -->
<c:catch var="에러">
    <% int result = 10 / 0; %> <!-- 의도적 에러 발생 -->
</c:catch>
<c:if test="${not empty 에러}">
    <p>계산 중 에러가 발생했습니다: ${에러.message}</p>
</c:if>
<!-- 실행결과: 계산 중 에러가 발생했습니다: / by zero -->


    

<!-- 9. URL 생성 (url) -->
<c:url var="검색URL" value="/search">
    <c:param name="keyword" value="JSTL" />
    <c:param name="category" value="책" />
</c:url>
<a href="${검색URL}">검색하기</a>
<!-- 실행결과: <a href="/search?keyword=JSTL&category=책">검색하기</a> -->


    

<!-- 10. 외부 자원 가져오기 (import) -->
<c:import var="네이버홈" url="https://www.naver.com" />
<div class="preview">
    ${네이버홈}
</div>
<!-- 실행결과: 네이버 홈페이지 내용이 div 안에 로드됨 -->


    

<!-- 11. 변수 제거 (remove) -->
<c:remove var="사용자이름" />
<p>삭제 후 이름: <c:out value="${사용자이름}" default="이름이 삭제됨" /></p>
<!-- 실행결과: 삭제 후 이름: 이름이 삭제됨 -->


    

<!-- 12. 페이지 리다이렉트 (redirect) -->
<c:if test="${empty 세션유저}">
    <c:redirect url="/login.jsp">
        <c:param name="returnUrl" value="/main.jsp" />
    </c:redirect>
</c:if>
<!-- 실행결과: 비로그인 시 /login.jsp?returnUrl=/main.jsp 로 리다이렉트 -->
  • Formatting 태그 예제 
    <%@ taglib prefix="fmt" uri="http://java.sun.com/jsp/jstl/fmt" %>

<!-- 숫자 포맷팅 -->
<c:set var="금액" value="1234567" />
<fmt:formatNumber value="${금액}" type="currency" currencySymbol="₩" />

<!-- 날짜 포맷팅 -->
<c:set var="현재날짜" value="<%= new java.util.Date() %>" />
<fmt:formatDate value="${현재날짜}" pattern="yyyy년 MM월 dd일" />
  • SQL 태그 예제 
    <%@ taglib prefix="sql" uri="http://java.sun.com/jsp/jstl/sql" %>

<!-- 데이터베이스 연결 -->
<sql:setDataSource var="데이터소스" 
    driver="com.mysql.jdbc.Driver"
    url="jdbc:mysql://localhost:3306/테스트DB"
    user="사용자명" 
    password="비밀번호" />

<!-- 데이터 조회 -->
<sql:query var="결과" dataSource="${데이터소스}">
    SELECT * FROM 회원테이블 WHERE 나이 >= ?
    <sql:param value="20" />
</sql:query>
  • Functions 태그 예제 
    <%@ taglib prefix="fn" uri="http://java.sun.com/jsp/jstl/functions" %>

<!-- 문자열 처리 -->
<c:set var="문자열" value="안녕하세요 JSTL입니다" />
