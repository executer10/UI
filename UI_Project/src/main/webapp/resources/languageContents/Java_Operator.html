연산자 관련 용어 설명
용어 설명
단항 연산자 피연산자가 단 하나뿐인 연산자. 부호 연산자(+, -), 증감 연산자(++, --), 논리 부정 연산자(!)가 해당됨.
부호 연산자 양수 및 음수를 표시하는 +, -. boolean 타입과 char 타입을 제외한 나머지 기본 타입에 사용 가능.
증감 연산자 변수의 값을 1 증가(++)시키거나 1 감소(--)시키는 연산자. boolean 타입을 제외한 모든 기본 타입의 피연산자에 사용 가능.
논리 부정 연산자(!) true를 false로, false를 true로 변경. boolean 타입에만 사용 가능.
이항 연산자 피연산자가 2개인 연산자. 산술 연산자, 문자열 결합 연산자(+), 비교 연산자, 논리 연산자, 대입 연산자 등이 해당됨.
문자열 결합 연산자(+) 문자열을 서로 결합하는 연산자. 피연산자 한쪽이 문자열이면 다른 피연산자를 문자열로 변환하고 서로 결합함.
비교 연산자 피연산자의 대소(<, <=, >, >=) 또는 등등(==, !=)을 비교해서 true/false를 산출. 주로 조건문(if), 반복문(for, while)에서 실행 흐름을 제어할 때 사용.
삼항 연산자 3개의 피연산자를 필요로 하는 연산자(?:). 조건식에 따라 콜론(:) 앞뒤의 피연산자가 선택됨.

new 연산자
정의 Java에서 객체를 생성하고 메모리를 할당하는 데 사용되는 연산자 입니다.
주요 기능
  • 클래스의 새 인스턴스를 생성합니다.
  • 객체에 필요한 메모리를 동적으로 할당합니다.
  • 객체의 생성자를 호출합니다.
사용법 ClassName objectName = new ClassName(parameters);
예시 // 매개변수가 없는 생성자 호출
Car myCar = new Car();

// 매개변수가 있는 생성자 호출
Student newStudent = new Student("John", 20);

// 배열 생성
int[] numbers = new int[5];
메모리 관리
  • 객체는 힙(Heap) 메모리에 생성됩니다.
  • 더 이상 참조되지 않는 객체는 가비지 컬렉터에 의해 자동으로 메모리에서 제거됩니다.
주의사항
  • 객체 생성 실패 시 OutOfMemoryError가 발생할 수 있습니다.
  • 불필요한 객체 생성은 메모리 낭비를 초래할 수 있으므로 주의해야 합니다.
