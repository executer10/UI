파이썬 기본 문법

제목

  • 변수와 자료형
  • 입력과 출력
  • 연산자와 문자열 슬라이싱
  • 조건문
  • 반복문
  • 함수

세부 정보 정의

  1. 변수와 자료형
    • 파이썬 변수의 특징 : 파이썬의 변수는 데이터 타입을 명시하지 않아도 됩니다. 변수에 값을 할당하면 자동으로 데이터 타입이 결정됩니다.
    • 데이터 타입
      • 자료형 : 정수형, 실수형, 문자열형, 부울형 등 다양한 데이터 타입을 지원합니다.
      • 실수형: 소수점을 포함한 숫자를 나타냅니다. 예: 3.14
      • 문자열형: 문자들의 집합으로, 따옴표로 감싸서 표현합니다. 예: "Hello"
      • 부울형: 참(True) 또는 거짓(False)을 나타내는 데이터 타입입니다.
    • 강제 형 변환: 데이터 타입을 명시적으로 변환하는 방법입니다. 예를 들어, 정수를 문자열로 변환할 수 있습니다.
  2. 입력과 출력
    • 입력: 사용자로부터 데이터를 입력받는 방법입니다. input() 함수를 사용합니다.
    • 출력: 데이터를 화면에 출력하는 방법입니다. print() 함수를 사용합니다.
    • 출력 서식: 출력할 데이터를 특정 형식으로 표현하는 방법입니다. 예: print(f"안녕하세요, {name}님!")
  3. 연산자
    • 산술 연산자: 덧셈, 뺄셈, 곱셈, 나눗셈 등을 수행하는 연산자입니다. 예: +, -, *, /
    • 비교 연산자: 두 값을 비교하여 참 또는 거짓을 반환하는 연산자입니다. 예: ==, !=, <, >
    • 논리 연산자: 논리적인 판단을 수행하는 연산자입니다. 예: and, or, not
  4. 문자열 슬라이싱
    • 문자열 인덱스: 문자열의 각 문자는 인덱스를 통해 접근할 수 있습니다. 인덱스는 0부터 시작합니다.
    • 문자열 인덱스 예시: 문자열이 'PYTHON'인 경우, P는 인덱스 0, Y는 인덱스 1입니다.
    • 문자열 슬라이싱: 문자열의 일부분을 추출하는 방법입니다. 예: text[1:4]는 'YTH'를 반환합니다.
  5. 조건문
    • 사용 형식: 특정 조건에 따라 다른 코드를 실행하는 방법입니다. if, elif, else 키워드를 사용합니다.
  6. 반복문
    • for
      • 반복 범위가 숫자인 경우: 지정된 범위 내에서 반복합니다. 예: for i in range(5):
      • 반복 범위가 리스트인 경우: 리스트의 각 요소에 대해 반복합니다. 예: for item in list:
    • while: 조건이 참인 동안 반복합니다. 예: while condition:
    • 반복문에서 사용하는 continue/break/else
      • continue: 현재 반복을 건너뛰고 다음 반복을 시작합니다.
      • break: 반복문을 종료합니다.
      • else: 반복문이 정상적으로 종료되었을 때 실행됩니다.
  7. 함수
    • 함수의 정의: 특정 작업을 수행하는 코드 블록을 정의하는 방법입니다. def 키워드를 사용합니다.

특징

  1. 파이썬은 간결한 문법을 가지고 있어 초보자에게 적합합니다.
  2. 다양한 데이터 타입을 지원하여 유연한 프로그래밍이 가능합니다.
  3. 강력한 문자열 처리 기능을 제공합니다.

사용 이유

  1. 초보자도 쉽게 배울 수 있는 간단한 문법
  2. 다양한 라이브러리와 프레임워크 지원
  3. 다양한 분야에서의 활용 가능성

예제 구문

# 1. 변수와 자료형
# 기본 자료형
number = 42          # 정수형
pi = 3.14           # 실수형
text = "Hello"      # 문자열형
is_valid = True     # 부울형

# 강제 형변환
str_num = str(number)        # 정수 → 문자열: "42"
int_str = int("100")        # 문자열 → 정수: 100
float_str = float("3.14")   # 문자열 → 실수: 3.14


# 2. 입력과 출력
name = input("이름을 입력하세요: ")
age = int(input("나이를 입력하세요: "))

# 다양한 출력 방식
print("일반 출력:", name)
print(f"f-string 출력: {name}님은 {age}세입니다.")
print("format 출력: {}님은 {}세입니다.".format(name, age))
print("%-formatting: %s님은 %d세입니다." % (name, age))


# 3. 연산자
# 산술 연산자
a, b = 10, 3
print(f"덧셈: {a + b}")        # 13
print(f"뺄셈: {a - b}")        # 7
print(f"곱셈: {a * b}")        # 30
print(f"나눗셈: {a / b}")      # 3.333...
print(f"몫: {a // b}")         # 3
print(f"나머지: {a % b}")      # 1
print(f"제곱: {a ** b}")       # 1000

# 비교 연산자
x, y = 5, 10
print(x == y)    # False
print(x != y)    # True
print(x < y)     # True
print(x >= y)    # False

# 논리 연산자
p, q = True, False
print(p and q)   # False
print(p or q)    # True
print(not p)     # False


# 4. 문자열 슬라이싱
text = "Python Programming"
print(text[0])           # 'P'
print(text[7:])         # 'Programming'
print(text[:6])         # 'Python'
print(text[7:11])       # 'Prog'
print(text[::2])        # 'Pto rgamn'
print(text[::-1])       # 'gnimmargorP nohtyP'


# 5. 조건문
score = 85
if score >= 90:
    print("A")
elif score >= 80:
    print("B")
elif score >= 70:
    print("C")
else:
    print("D")


# 6. 반복문
# for 반복문
for i in range(3):
    print(i, end=' ')  # 0 1 2

fruits = ['apple', 'banana', 'cherry']
for fruit in fruits:
    print(fruit, end=' ')  # apple banana cherry

# while 반복문
count = 0
while count < 3:
    print(count, end=' ')
    count += 1  # 0 1 2

# continue, break, else 활용
for i in range(5):
    if i == 2:
        continue    # 2를 건너뜀
    if i == 4:
        break       # 4에서 종료
    print(i, end=' ')  # 0 1 3
else:
    print("반복 완료")  # break로 종료되어 실행되지 않음


# 7. 함수
def calculate_area(width, height):
    """사각형의 넓이를 계산하는 함수"""
    return width * height

def greet(name, greeting="안녕하세요"):
    """인사말을 출력하는 함수"""
    return f"{greeting}, {name}님!"

# 함수 호출
area = calculate_area(5, 3)
message = greet("홍길동")
print(area)      # 15
print(message)   # 안녕하세요, 홍길동님!

not x가 아니다.(즉, x가 참이면 거짓, x가 거짓이면 참)
