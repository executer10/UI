세부 내용

1. 기본 개념

1.1 표현 언어의 정의

JSP 표현 언어(Expression Language)는 JSP 페이지에서 자바 코드 사용을 최소화하고 데이터를 쉽게 출력하기 위한 스크립트 언어입니다.

1.2 사용 목적

  • JSP 페이지 작성을 단순화
  • 자바 코드와 마크업의 분리
  • 데이터 접근 단순화
  • 타입 변환 자동화
2. 핵심 기능

2.1 데이터 접근 방식

  • 스코프별 데이터: ${스코프.속성명}
  • 폼 데이터: ${param.파라미터명}
  • 객체 속성: ${객체.속성명}

2.2 내장 객체 활용

  • pageScope, requestScope, sessionScope, applicationScope
  • param, paramValues
  • header, headerValues
  • cookie
  • initParam
  • pageContext

2.3 연산자 사용법

  • 산술 연산자: +, -, *, /, %
  • 비교 연산자: ==, !=, <, >, <=, >=
  • 논리 연산자: &&, ||, !
  • 조건 연산자: ? :
  • empty 연산자
3. 실전 활용

3.1 데이터 출력

  • 단순 변수 출력
  • 객체 속성 출력
  • 컬렉션 데이터 출력

3.2 객체 처리

  • 자바빈즈 객체 접근
  • List, Map 등 컬렉션 처리
  • 커스텀 객체 활용

3.3 조건부 로직

  • 삼항 연산자 활용
  • empty 검사
  • null 처리
  • 조건부 데이터 출력

특징

  1. 간편한 데이터 접근: 복잡한 자바 코드 없이 데이터 출력 가능
  2. 가독성 향상: HTML과 자바 코드의 분리로 코드 관리 용이
  3. 자동 형변환: 데이터 타입 자동 변환 지원
  4. 널 처리: null 값에 대한 안전한 처리 제공
  5. 연산자 지원: 다양한 연산 기능 제공

예제 구문

1. 기본 데이터 처리 
<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<%@ page import="java.util.*" %>

<% 
    // 1단계: 기본 데이터 설정
    request.setAttribute("사용자_이름", "홍길동");
    request.setAttribute("나이", 25);
    
    // 실행 결과:
    // - request 영역에 사용자_이름="홍길동" 저장됨
    // - request 영역에 나이=25 저장됨
%>

<!-- 기본 데이터 출력 예시 -->
이름: ${사용자_이름}  <!-- 출력결과: 홍길동 -->
나이: ${나이}        <!-- 출력결과: 25 -->
2. 컬렉션 데이터 처리 
<% 
    // 2단계: 리스트 데이터 설정
    List<String> 취미목록 = Arrays.asList("독서", "운동", "여행");
    request.setAttribute("취미", 취미목록);
    
    // 실행 결과: request 영역에 취미=[독서, 운동, 여행] 저장됨
    
    // 3단계: 맵 데이터 설정
    Map<String, String> 정보 = new HashMap<>();
    정보.put("직업", "개발자");
    정보.put("이메일", "hong@example.com");
    request.setAttribute("상세정보", 정보);
    
    // 실행 결과: request 영역에 상세정보={직업=개발자, 이메일=hong@example.com} 저장됨
%>

<!-- 컬렉션 데이터 출력 예시 -->
<h4>리스트 데이터 접근</h4>
첫 번째 취미: ${취미[0]}  <!-- 출력결과: 독서 -->
두 번째 취미: ${취미[1]}  <!-- 출력결과: 운동 -->

<h4>맵 데이터 접근</h4>
직업: ${상세정보.직업}     <!-- 출력결과: 개발자 -->
이메일: ${상세정보.이메일} <!-- 출력결과: hong@example.com -->
3. 조건부 처리와 null 처리 
<!-- 조건부 출력 예시 -->
<h4>삼항 연산자 활용</h4>
성인 여부: ${나이 >= 20 ? '성인' : '미성년자'}
<!-- 출력결과: 성인 -->

<h4>null 값 처리</h4>
없는 데이터: ${없는데이터 != null ? 없는데이터 : '데이터 없음'}
<!-- 출력결과: 데이터 없음 -->

<h4>empty 연산자 활용</h4>
취미 존재 여부: ${empty 취미 ? '취미 없음' : '취미 있음'}
<!-- 출력결과: 취미 있음 -->
