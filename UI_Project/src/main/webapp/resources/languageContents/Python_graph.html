세부 내용

  1. 맷플롯립이란?
    • 맷플롯립은 파이썬에서 데이터 시각화를 위한 라이브러리입니다.
  2. 맷플롯립으로 그래프 그리기
    1. 선그래프: plot()
      • 사용법: plot(x값, y값, 선 스타일 문자열, marker='마커기호', label='그래프이름')
      • y값은 필수고 나머지는 생략 가능
    2. 산점도 그래프: scatter()
      • 사용법: scatter(x값, y값, c='색 문자' 혹은 '색깔 문자열', marker='마커기호', label='그래프 이름')
    3. 막대 그래프: bar()/barh()
      • 사용법: bar(x값, y값, c='색 문자' 혹은 '색깔 문자열')
      • 범주별 데이터를 나타내는 세로 막대 그래프를 생성하며, x값은 문자열도 가능하다.
      • 사용법: barh(x값, y값, c='색 문자' 혹은 '색깔 문자열')
      • 범주별 데이터를 나타내는 가로 막대 그래프를 생성하며 y값은 문자열도 가능하다.
    4. 빈도 그래프(히스토그램): hist()
      • 사용법: hist(x값, bins=x값 구간 개수, color='색 문자' 혹은 '색깔 문자열', align='left'/'mid'/'right' 중 하나)
      • x값으로 전달된 데이터의 개별 항목들이 각 구간에 포함되는 개수를 그래프로 나타낸다.
    5. 그래프 제목, x축, y축 제목 지정: title(), xlabel(), ylabel()
      • 사용법: title('문자열')
      • 그래프의 제목을 표시한다.
      • 사용법: xlabel('문자열')
      • x축 제목을 표시한다.
      • 사용법: ylabel('문자열')
      • y축 제목을 표시한다.
    6. 한 화면에 여러 개 그래프 나타내기: subplots()
      • 사용법: subplots(행 개수, 열 개수, figsize=(가로 길이, 세로 길이))
      • 여러 개의 그래프를 한 화면에 그리도록 화면을 분할한다.
    7. 맷플롯립(matplotlib)을 사용할 때 한글 출력하는 방법
      • 주피터 노트북을 사용한 경우
      • 코랩을 사용하는 경우
  3. 시본이란?
    • 시본은 파이썬에서 데이터 시각화를 위한 고급 라이브러리입니다.
  4. 시본으로 그래프 그리기
    1. 산점도 그래프: scatterplot()
      • 사용법: scatterplot(x='x축에 표시할 칼럼명', y='y축에 표시할 칼럼명', hue='데이터 구분 칼럼명', data=데이터프레임)
      • x,y에 있는 두 데이터 간의 상관관계를 좌표평면의 점으로 나타내는 산점도 그래프를 그린다.
    2. 범주별 데이터 개수를 표시하는 막대 그래프: counterplot()
      • 사용법: counterplot(x='x축에 범주로 표시할 칼럼명', hue='데이터 구분 칼럼명', data=데이터프레임)
      • 범주별 데이터 개수를 나타내는 막대 그래프를 그린다.
    3. 범주별 항목값 평균을 표시하는 막대 그래프: barplot()
      • 사용법: barplot(x='x축에 표시할 칼럼명', y='y축에 표시할 칼럼명', hue='데이터 구분 칼럼명', data=데이터프레임)
      • x값에 대한 y값 평균 막대 그래프로 표시한다.
    4. 히스토그램: histplot()
      • 사용법: histplot(x='x축에 표시할 칼럼명', kde=True/False, data=데이터프레임)
      • x로 전달된 데이터의 개별 항목들이 각 구간에 포함되는 개수 분포를 나타내는 히스토그램을 그린다.
    5. 숫자값을 갖는 칼럼들의 상관관계를 보여주는 그래프: pairplot()
      • 사용법: pairplot(데이터프레임, hue='범주를 갖는 칼럼명')
      • hue를 지정하면 범주별로 구분해서 칼럼들 간의 상관관계를 표시한다.
    6. 산점도와 추세선(회귀선)을 동시에 보여주는 그래프: regplot()
      • 사용법: regplot(x='x축에 표시할 칼럼명', y='y축에 표시할 칼럼명', data=데이터프레임)
      • x,y에 있는 두 데이터 간의 상관관계에 대해서 좌표평면의 점과 추세선(회귀선)을 동시에 나타내는 그래프를 그린다.
    7. 집계한 값에 따라서 색깔을 표시하는 그래프(히트맵): heatmap()
      • 사용법: heatmap(data=2차원 행렬 형태로 전달된 데이터, annot=True/False, fmt='파이썬 서식문자열', cbar=True/False)

특징

  1. 맷플롯립과 시본은 다양한 그래프를 그릴 수 있는 강력한 기능을 제공합니다.
  2. 사용자가 원하는 스타일로 그래프를 커스터마이징할 수 있습니다.

사용 이유

  1. 데이터 시각화를 통해 복잡한 데이터를 쉽게 이해할 수 있습니다.
  2. 다양한 그래프 스타일을 통해 데이터를 효과적으로 표현할 수 있습니다.

import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

# 데이터프레임 생성
data = {
    '이름': ['홍길동', '이몽룡', '성춘향', '변학도'],
    '수학': [90, 85, 88, 95],
    '영어': [85, 90, 92, 88],
    '과학': [95, 80, 85, 90]
}

df = pd.DataFrame(data)

# 데이터프레임 출력
print(df)

# 시본 스타일 설정
sns.set_style('whitegrid')

# 막대 그래프 그리기
plt.figure(figsize=(10, 6))
sns.barplot(x='이름', y='수학', data=df)
plt.title('학생별 수학 점수')
plt.xlabel('학생 이름')
plt.ylabel('수학 점수')
plt.show()

# 산점도 그래프 그리기
plt.figure(figsize=(10, 6))
sns.scatterplot(x='수학', y='영어', data=df, hue='이름', style='이름', s=100)
plt.title('수학과 영어 점수의 상관관계')
plt.xlabel('수학 점수')
plt.ylabel('영어 점수')
plt.show()

# 히스토그램 그리기
plt.figure(figsize=(10, 6))
sns.histplot(df['과학'], bins=5, kde=True)
plt.title('과학 점수 분포')
plt.xlabel('과학 점수')
plt.ylabel('빈도')
plt.show()
